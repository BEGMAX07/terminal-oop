Как собрать и запустить

Сборка и запуск:
g++ main.cpp -o terminal.exe
Найти файл terminal.exe в папке проекта и запустить его.

Какие команды поддерживаются:

help - Выводит список доступных команд.
exit - Завершает работу программы.
ls - Показывает содержимое текущей директории.
cd - Переходит в другую директорию.
cat - Показывает содержимое текстового файла.
touch - Создаёт новый файл в текущей директории.
mkdir - Создаёт новую директорию.
rm - Скрывает файл или директорию вместо настоящего удаления.
restore - Восстанавливает ранее скрытую сущность.
mv - Переименовывает файл или директорию.
edit - Изменяет содержимое файла.

Какие авторитативные решения были выбраны в архитектуре:

1. Где определяется свойство скрытости
Флаг bool isHidden хранится в базовом классе Entry.
2. Где происходит фильтрация скрытых файлов при ls
Фильтрация выполняется в методе Directory::print().
3. Где реализован поиск сущности по имени
Поиск реализован в классе Directory (методы findChild, findChildIncludingHidden).
4. Как организован текущий путь (cwd)
Текущая директория хранится как указатель Directory* cwd в классе Terminal.
5. Как устроен переход между директориями
Переход реализован через указатель Directory* parent в классе Directory.
6. Где реализован полиморфизм
Полиморфизм реализован через виртуальные методы в классе Entry: print(), edit(), isDirectory(), hide(), restore().
